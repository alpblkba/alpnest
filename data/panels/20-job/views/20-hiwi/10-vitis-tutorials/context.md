# Vitis Tutorials Context

This section tracks the broad Vitis tutorial repository provided for HiWi
preparation.

Repository:

https://github.com/Xilinx/Vitis-Tutorials

Use this as ecosystem orientation. Do not treat it as a requirement to learn every
Vitis subsystem before the Monday meeting.

The current focus is HLS-based accelerator work for embedded processors, likely
connected to legacy lab rewriting. Therefore, prioritize the parts that explain
how HLS projects are structured, how they are built, and how generated RTL/IP can
be consumed downstream.
